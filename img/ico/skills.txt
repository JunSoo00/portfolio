HTML/CSS 시멘틱 마크업과 반응형 설계, 
JavaScript·jQuery로의 인터랙션 구현, 
React로의 컴포넌트화에 강점.

핵심 스킬 & 역량 (KR)

HTML

접근성·SEO를 고려한 시멘틱 마크업(landmark, heading 구조, aria 속성)

재사용 가능한 컴포넌트형 마크업 패턴 설계 (BEM/Atomic 등 네이밍 합의)

폼/테이블/미디어 요소의 표준 속성 최적 활용

CSS

반응형 레이아웃(Grid/Flex), 다크모드/테마 토큰, CSS 변수

애니메이션/트랜지션으로 마이크로 인터랙션 구현

크로스 브라우징 전략(Autoprefixer, 폴백, 지원 범위 정의) 및 성능 최적화(페인트/레이아웃 비용 최소화)

JavaScript

ES6+ 문법 숙련(모듈, 비구조화, async/await), 이벤트·DOM 제어

API 연동(fetch/axios), 유효성 검증, 상태/뷰 로직 분리로 가독성/테스트성 향상

성능 최적화(디바운스/스로틀, 지연 로딩, 번들 사이즈 관리 개념 이해)

jQuery

레거시/간단한 랜딩·사내 툴에서 빠른 UI 프로토타이핑

플러그인 커스터마이징(슬라이더, 모달, 폼 검증 등), 기존 코드 안정적 유지보수

React

함수형 컴포넌트/훅 기반 UI 설계, 상태 관리 최소화 전략

컴포넌트 분해/합성, props·context 패턴, 비동기 데이터 패칭

코드 스플리팅/지연 로딩로 초기 구동 최적화, 접근성 속성/키보드 내비게이션 적용

Figma

와이어프레임→하이파이 시안 제작, Auto Layout/Variants로 디자인 시스템 구성

디자인 토큰(색/타이포/스페이싱) 정의, 개발자 핸드오프(Measure/Spec) 정리

프로토타입 인터랙션과 사용자 플로우 생성 및 관리

Photoshop

웹용 이미지 편집(리터칭, 보정, 합성), 에셋 내보내기 최적화(용량/해상도/포맷)

배너/썸네일/히어로 이미지 제작, 레이어/스마트오브젝트 워크플로우

Premiere Pro

짧은 프로모션/튜토리얼 영상 제작, 컷 편집/자막/기본 효과

루마/오디오 정리, 플랫폼별 아웃풋(YouTube/인스타 Reels 등) 세팅


HTML/CSS: 시멘틱 마크업, 반응형(Fluid/Grid/Flex), 접근성 기초, 컴포넌트 단위 스타일링

JavaScript(ES6+): DOM/이벤트, 비동기 통신(Ajax), 상태 변화에 따른 UI 업데이트

React: JSX/Virtual DOM, 함수형·클래스형 컴포넌트, props/state, 리스트 렌더링(map/key), 라이프사이클/훅 개념, 스타일링 & 반응형

퍼블리싱/UX 협업: 웹사이트 제작 프로세스, 최신 트렌드 반영, 실습 산출물 포트폴리오화
